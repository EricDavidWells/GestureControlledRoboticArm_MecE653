# HACKED2019
Create a classification / regression based algorithm based off of FSR's and muscle activity.

## Classification Labels
* 0 --> relax
* 1 --> flexion
* 2 --> extension
* 3 --> fist
* 4 --> supnation
* 5 --> pronation
* 6 --> middle finger down
* 7 --> ring finger down
* 8 --> pointer finger down
* 9 --> light fist left
* 10 --> light fist right
